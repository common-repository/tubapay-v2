== Description ==

Plugin do płatności abonamentowych TubaPay.

== Installation ==

1. Wgraj rozpakowany plugin do folderu /wp-content/plugins/
2. Idź do zakładki **Wtyczki** i aktywuj plugin.

 lub

1. Wgraj plik zip ręcznie w zakładce **Wtyczki**

Po zakończeniu instalacji wtyczki TubaPay zgodnej z jedną z powyższych instrukcji upewnij się, że w ustawieniach woocommerce zamówienia cząstkowe są aktywne, aby fakturować swoich Klientów.
